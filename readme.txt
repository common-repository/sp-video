== Description ==
SP-Video Plugin is a plugin that alow user to post a video from youtube without using embed code. 
It`s very easy to use: 
1 - Copy link of your youtube video 
2 - Insert "[video] Youtube link [/video]" in your post.

== Installation ==

1. Upload sp-video folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

1, 2, 3: You're done!

== Frequently Asked Questions ==

= Why was this plugin created? =

It is more simple to just copy/paste YouTube link and not to search for embed code.
